This project focuses on developing a machine learning classification model to help the mobile carrier Megaline recommend the most suitable new plan (Smart or Ultra) to subscribers currently using legacy plans.

Using customer behavior data from subscribers who have already switched to the new plans, the goal was to build a model capable of accurately predicting which plan best fits a user’s usage patterns. Since the data preprocessing step was completed beforehand, the project concentrated on model selection, training, and evaluation.

The objective was to achieve the highest possible accuracy, with a required minimum threshold of 0.75. Model performance was validated using a separate test dataset to ensure reliable and generalizable results.

This project demonstrates practical application of supervised learning techniques for real-world business decision-making in the telecommunications industry.
